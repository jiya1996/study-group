## 🎯 阶段二：认知偏误专题（2-3周）

### 投资者常见偏误清单

| 偏误 | 表现 | 典型场景 | 应对策略 |
|------|------|----------|----------|
| 损失厌恶 | 亏损痛苦是盈利快乐的2倍 | 熊市割肉 | 框架效应、淡化的时间 |
| 确认偏误 | 只看支持自己观点的信息 | 死多头/死空头 | 多角度论证、预设反对意见 |
| 锚定效应 | 过度依赖第一个信息 | "以前300块" | 重新锚定、动态参考点 |
| 后见之明 | 事后认为"早就知道" | 拍大腿 | 记录事前预测、概率思维 |
| 过度自信 | 高估自己的能力 | 频繁交易 | 记录决策日志、对冲 |
| 处置效应 | 卖盈利留亏损 | 赚就跑、亏就套 | 设定卖出规则 |
| 羊群效应 | 跟随大众决策 | 追热点 | 独立思考、逆向投资 |
| FOMO | 害怕错过 | 牛市顶峰入场 | 等待冷却、设定入场规则 |

### 实践练习

1. 每周记录自己的 3 个投资决策，分析背后的心理
2. 复盘最近一次 "拍大腿" 事件，找出触发偏误的信号

## 🎯 阶段三：市场情绪周期（2周）

### 关键概念

- **情绪周期**：恐慌 → 绝望 → 怀疑 → 观望 → 乐观 → 兴奋 → 贪婪 → 泡沫 → 焦虑 → 恐慌
- ** VIX 指数**：恐慌指数，量化市场情绪
- **贪婪恐惧指数**：多指标综合（CNN Fear & Greed Index）

### 学习资源

- 《周期》-霍华德·马克斯
- AQR Capital 研究报告
- market-psychology.com

## 🎯 阶段四：助推策略实践（持续）

### 什么是助推（Nudge）

不强制改变选择，而是通过设计选择架构，让人们做出更好决策。

### 投资场景的助推应用

| 场景 | 助推策略 |
|------|----------|
| 频繁查看账户 | 设置冷静期提醒 |
| 追涨杀跌 | 展示历史"后悔数据" |
| 过度交易 | 交易成本可视化 |
| 账户亏损 | 强调长期复利力量 |
| FOMO 入场 | 逆向信号提醒 |
