# 投资心理学研究 - BudVest

## 研究目的

这些资料服务于"投资心理观"整体战略的两个出口：

- **Skill 产品（投资情绪急诊室）**：理论→情绪识别逻辑→认知偏差映射→CBT 干预策略，直接写入 Skill 的 system prompt
