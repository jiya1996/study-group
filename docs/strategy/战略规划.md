# 投资心理观 — 战略规划

## 核心转向

BudVest 从"独立 App + 7 大师人格"转向三件事互相喂养的飞轮：

```
"投资心理观"视频账号（引流 + 建立信任）
        ↓ 用户认可你的专业度
"投资心理陪伴" Skill（产品变现）
        ↑ 内容素材来自
投资心理学 & 行为金融学研究（知识地基）
```

## 为什么放弃旧方案

- 堆砌功能（7 大师 + 投资组合 + RAG 知识库 + 市场数据），0 用户
- 独立 App 获客成本高，用户要下载/注册才能用
- Skill 形态寄生在用户已有的 AI 工具链里（Claude Code、OpenClaw 等），零获客摩擦

## 三条线

### 1. Skill：投资情绪急诊室（产品）

详见 [投资情绪急诊室-设计](../product/投资情绪急诊室-设计.md)

### 2. 视频账号：投资心理观（分发）

**定位：** 用投资心理学和行为金融学的视角，帮散户看清自己的情绪陷阱

**平台策略：**

| 优先级 | 平台 | 策略 |
|--------|------|------|
| 主力 | 小红书 | 投资焦虑人群密度最高，短视频+图文，互动率高，最容易起号 |
| 主力 | 抖音 | 流量天花板最高 |
| 分发 | B站 | 同一条视频顺手发 |
| 分发 | 视频号 | 同一条视频顺手发 |
| 暂缓 | YouTube/Twitter | 英文或双语内容再考虑，现阶段不分心 |

**核心原则：** 一条视频只生产一次，多平台分发。不为每个平台单独做内容。

**内容与 Skill 的联动：**
- 视频讲理论（如"散户常见的 5 种情绪陷阱"）
- 结尾引导"试试我的 AI 情绪急诊室"
- 用户使用 Skill 的真实案例（脱敏后）反哺视频内容

### 3. 投资心理学研究（知识地基）

详见 [研究索引](../research/研究索引.md)

已有 17 篇核心论文导入 NotebookLM，覆盖前景理论、行为偏误、情绪与决策三大方向。

研究产出同时服务于：
- Skill 的 system prompt 设计（理论→干预策略）
- 视频内容脚本（理论→通俗表达）

## 执行节奏

| 阶段 | 时间 | 交付物 |
|------|------|--------|
| MVP | 第 1 周 | 投资情绪急诊室 Skill v0.1 可用 |

| 迭代 | 第 5 周起 | 根据用户反馈优化 Skill；根据数据调整内容方向 |
