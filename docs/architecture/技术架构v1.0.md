# BudVest 技术架构 v1.0 版

> 架构概览 · 2026-1-22

---

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Next.js 15 + React 18 + TypeScript + TailwindCSS |
| 后端 | Next.js API Routes (Serverless) |
| 数据库 | Supabase (PostgreSQL + pgvector) |
| AI | Claude Agent SDK + OpenAI (GPT-4o-mini) |
| 缓存 | Upstash Redis + localStorage |
| 部署 | Vercel + PWA |

---

## 核心模块

```
app/
├── page.tsx          # 主页 (4个Tab切换)
├── api/market/       # 市场数据 API (10+端点)
├── api/portfolio/    # 投资组合 API
└── api/chat-claude/  # AI 对话 API (主)

lib/
├── types.ts          # 类型定义
├── rag.ts            # RAG 知识检索
├── memory.ts         # 用户记忆系统
├── market-data.ts    # 三层数据访问
└── stock-utils.ts    # 股票工具

components/
├── MarketTab.tsx     # 市场数据
├── CompanionTab.tsx  # AI 伴侣
├── PortfolioTab.tsx  # 持仓管理
└── ProfileTab.tsx    # 个人设置
```

---

## AI 系统特色

- **7 大师人设**: 巴菲特、索罗斯、芒格、达利欧、林奇、木头姐、心理教练
- **情绪识别**: anxious / panic / angry / greedy / calm
- **安全机制**: 完全禁止投资建议（买入/卖出/加仓等）
- **RAG 知识库**: 大师语录、投资案例、投资原则

---

## 数据流

```
用户输入 → 意图识别 → RAG检索 → 大师人设注入 → LLM生成 → 安全过滤 → 响应
```
