## 目标

打造 **BudVest** —— 一个基于行为金融学的投资心理陪伴产品。

核心假设：散户的投资亏损，更多来自情绪驱动的冲动决策，而非信息不足。我们用 AI 帮投资者在冲动操作前"暂停一下"。

## 快速开始

**试用投资情绪急诊室 Skill：**

将 `skills/investment-emotion-er/` 复制到你的 `~/.claude/skills/` 目录，Claude Code 会自动识别。当你表达投资情绪时（如"跌麻了想割肉"），Skill 会自动触发。
