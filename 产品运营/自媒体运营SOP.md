# 第一部分：「小何学AI」

## 一、频道定位

### 1.1 一句话定位

**一个非技术人，从"被AI搞懵"到"让AI为我所用"的真实成长记录。**

### 1.2 目标读者

- 非技术背景的职场人（产品经理、运营、业务人员等）
- 对 AI 感兴趣但不知道从何下手的人
- 用过 ChatGPT 但停留在"聊天"层面，想深度使用的人

### 1.3 差异化优势

市面上的 AI 科普内容主要分两类：

- **技术博主**：写给开发者，门槛高，非技术人看不懂
- **泛科普博主**：内容太浅，停留在"10个ChatGPT技巧"

「小何学AI」的独特定位：**有深度但不需要编程背景**。作者本人是产品经理，用真实的踩坑经历和实战案例输出方法论，读者能直接复用。

### 1.4 核心竞争力

- **讲故事的能力**：能把复杂的 AI 概念说明白——这是贯穿所有内容的风格标签
- **真实经历不可复制**：每一篇都来自亲身实践，不是搬运和二手信息
- **产品经理视角**：关注"AI能帮我解决什么问题"，而不是"AI的技术原理是什么"

---

## 二、内容框架：四阶段成长路径

所有选题围绕一个非技术人学 AI 的真实成长路径展开，分为四个阶段：

### 阶段一：认知破冰——"AI到底是什么"

帮读者建立基本认知，消除恐惧和误解。

| 选题方向 | 示例 |
|---------|------|
| AI 核心概念科普 | 大模型、Agent、MCP 是什么？用一顿火锅讲明白 |
| 解读厂商发布（适度） | Anthropic/OpenAI 又发新东西了，跟我有什么关系？ |

> ⚠️ 注意：概念科普和热点解读不是主力内容，适度穿插即可。这类内容竞争激烈且时效性强。

### 阶段二：踩坑觉醒——"我试了，但翻车了"

分享真实的失败和反思，建立读者信任。

| 选题方向 | 示例 |
|---------|------|
| 使用 AI 工具的翻车记录 | 用 AI 指挥千军万马，自己却成了最大的瓶颈（已发布） |
| 从放弃到回头的转折点 | 我为什么放弃了 Claude Code，又为什么捡回来 |

> 🏆 这是你的护城河内容之一。真实的踩坑经历别人抄不走。

### 阶段三：方法论建立——"我摸索出了一套用法"

从实战中提炼可复用的方法。

| 选题方向 | 示例 |
|---------|------|
| 提示词技巧 | 不会写 prompt？试试这三个思考框架 |
| AI 学习方法 | 如何让 AI 变成你的私人老师 |
| 工具使用妙招 | 每个大模型的"隐藏技能"：我的使用心得 |

> 📌 方法论要从实战中"长出来"，不要凭空编造。每篇方法论背后都应该有一个真实场景。

### 阶段四：实战应用——"AI真的在帮我干活了"

展示 AI 在真实工作场景中的应用。

| 选题方向 | 示例 |
|---------|------|
| 用 AI 搭产品 | 零代码基础，我用 Claude Code 搭了一个投资工具 |
| 用 AI 做自动化 | 我让 OpenClaw 每天帮我追热点、存灵感、写初稿 |
| 用 AI 做内容 | 这篇文章的初稿，就是 AI 帮我写的（附对比） |

> 🏆 这是你的另一个护城河。实战案例是最有说服力的内容。

---

## 三、内容策略

### 3.1 两条内容线

| 内容线 | 精力占比 | 说明 |
|-------|---------|------|
| **主线：AI 实战日记** | 70% | 写你真实在做的事——"我用 AI 干了什么、踩了什么坑、悟出了什么"。不需要额外找素材，顺手记录即可。 |
| **副线：小白方法论** | 30% | 从实战中提炼方法论。方法论从主线"长出来"，不独立找选题。 |

### 3.2 更新节奏

- **单篇篇幅**：1500-2000 字
- **核心原则**：不追求完美，追求持续。先跑起来，找到自己的自然节奏后再固定频率。

### 3.3 现阶段不做的事

- ❌ 模型横向测评
- ❌ 模型版本对比
- ❌ 厂商发布深度解读（偶尔穿插可以，不作为主线）
- ❌ 短视频内容（未来考虑，现阶段聚焦图文）

# 第二部分：「投资心理观」

## 一、频道定位

### 1.1 一句话定位

**用故事讲透投资心理学，帮你看见自己投资行为背后的心理陷阱。**

### 1.2 商业目的

为伴投（Budvest）产品预热，积累目标用户群。通过投资心理学内容建立信任，当伴投上线时，读者天然成为第一批用户。

### 1.3 目标读者

- 投资纯小白，刚开始接触投资理财
- 炒过股/买过基金但总是亏钱，不知道为什么
- 对"为什么我管不住自己的手"这类问题感兴趣的人

### 1.4 作者身份：同路人，不是老师

你自己是基金投资者，投资金额10万左右，没有股票经验，正在学习投资心理学。这不是劣势，而是核心优势——你跟读者是同一个身份。

定位宣言（可用于频道介绍或第一篇文章）：**"我不是投资高手，我是一个正在学习投资心理学的普通人。我把学到的东西用故事讲给你听。"**

这种坦诚比假装专家更能建立信任。读者会想："他跟我一样，他踩的坑我也在踩。"

### 1.5 差异化优势

市面上的投资心理学内容主要有两种：

- **学术派**：讲卡尼曼、塞勒的理论，专业但枯燥，小白看不下去
- **鸡汤派**：空喊"要克服贪婪和恐惧"，没有实操价值

「投资心理观」的独特定位：**用真实案例和历史故事讲投资心理学，同时记录自己作为普通投资者的真实心理变化**。读者在故事里不知不觉学到心理学原理，看完能对照自己的投资行为。

### 1.6 核心竞争力

- **讲故事的能力**：复杂的心理学概念通过故事说明白
- **同路人视角**：自己也是投资小白，身上的心理偏差就是最真实的素材
- **伴投产品联动**：内容不只是科普，而是通往产品体验的入口
- **7个大师人格**：伴投里的大师人格可以直接成为内容IP

### 1.6 与伴投的导流设计

内容到产品的转化路径：

```
读者看到一篇"巴菲特如何克服损失厌恶"的文章
    ↓
文末引导："如果你能随时问巴菲特怎么看你的持仓，你会怎么决策？"
    ↓
伴投产品入口（产品上线后）
```

> 📌 初期不需要硬推产品。先用内容建立信任和专业形象，等伴投上线时再加入导流钩子。

## 二、内容框架：四个系列

### 系列一：「你为什么亏钱」——心理陷阱系列（首先启动）

每篇讲一个投资心理偏差，用一个真实或历史案例引入。

**文章结构模板**：

1. **开头**：一个具体的投资场景（读者能代入的）
2. **故事**：一个真实/历史案例，展示这个心理陷阱如何导致亏损
3. **揭示**：这背后是什么心理学原理（用大白话解释）
4. **自测**：你有没有类似的行为？（让读者对照自己）
5. **一招**：一个简单的应对方法

**选题库（按心理偏差分类）**：

| 心理偏差 | 通俗标题示例 | 案例方向 |
|---------|------------|---------|
| 损失厌恶 | 为什么你舍不得割肉？ | 散户死扛亏损股的故事 |
| 羊群效应 | 别人都在买，我能不买吗？ | 2015年A股牛市散户蜂拥入场 |
| 锚定效应 | 你被"原价"骗了多少钱？ | 用买入价格当锚，忽略基本面变化 |
| 过度自信 | 连赚三次之后最危险 | 新手运气好连赢→加大仓位→巨亏 |
| 沉没成本 | 已经亏了这么多，卖了不就白亏了？ | 越补越套的经典陷阱 |
| 确认偏差 | 你只看到了你想看的 | 只关注支持自己判断的信息，忽略利空 |
| 禀赋效应 | 你的股票没你想的那么好 | 持有的股票总觉得比别的好 |
| 近因偏差 | 上周涨了所以明天也会涨？ | 用近期走势预测未来 |
| 心理账户 | 赚来的钱不是钱？ | 投资赚的钱更随意花掉/更激进投资 |
| 处置效应 | 赚了急着跑，亏了死不走 | 赢的时候太早卖出，输的时候太晚止损 |
| 框架效应 | 同一件事，换个说法你就上当了 | "亏损10%"vs"保住了90%" |
| 可得性偏差 | 你的判断被刷屏的新闻左右了 | 热点新闻驱动的非理性买入 |

> 📌 这个选题库有12+个选题，足够写三个月以上。每个心理偏差都是常青内容，不过时。

### 系列二：「大师怎么想」——大师心理模型系列

围绕伴投的7个大师人格展开，每篇讲一个大师的投资哲学，落脚点是他的"心理模型"。

**文章结构模板**：

1. **开头**：大师的一个经典投资决策场景
2. **故事**：他在关键时刻是怎么想的、怎么做的
3. **心理模型**：他克服了什么心理偏差、用了什么心理框架
4. **对比**：普通散户在同样场景下通常会怎么做
5. **启发**：你能从他身上学到什么（不是模仿他的操作，而是学他的思维方式）

**选题方向（对应伴投7个大师人格）**：

| 大师 | 核心心理模型 | 选题方向 |
|------|------------|---------|
| 巴菲特 | 逆向思维+极度耐心 | 当所有人恐惧的时候，他在想什么？ |
| 芒格 | 多元思维模型+避免愚蠢 | 比聪明更重要的是不犯蠢 |
| 格雷厄姆 | 安全边际+情绪隔离 | 把市场当一个情绪不稳定的合伙人 |
| 彼得·林奇 | 日常观察力+独立判断 | 你的超市购物车里藏着投资机会 |
| 索罗斯 | 反身性理论+承认错误 | 他最厉害的不是做对，而是认错够快 |
| 达利欧 | 系统思维+原则驱动 | 用原则代替情绪做决策 |
| 霍华德·马克斯 | 周期思维+第二层思考 | 所有人都知道的信息已经没有价值了 |

> 🏆 这个系列天然为伴投导流。读者对大师感兴趣→想跟大师"对话"→伴投产品入口。

### 系列三：「投资心理自测」——互动系列

给读者一个投资场景，让他们做选择，然后揭示背后的心理学原理。

**内容形式**：

- 图文版：设计2-3个场景选择题，文末揭晓并解读
- 未来视频版：口播+场景卡片，适合短视频传播
- 传播钩子：结尾加一句"转给你那个总是管不住手的朋友"

**示例选题**：

| 自测主题 | 场景设计 |
|---------|---------|
| 你是哪种投资性格？ | 5个场景判断你偏向激进/保守/跟风 |
| 你能扛住亏损吗？ | 模拟持仓下跌10%/20%/30%，你怎么选 |
| 你会被大V带跑吗？ | 同一条信息不同来源，你的判断会变吗 |

> 📌 互动内容传播性最强，但制作成本也最高。建议先把系列一和二跑稳，再穿插系列三。

### 系列四：「我的投资心理日记」——个人真实记录

记录你自己作为普通基金投资者的真实心理变化。这是你独有的护城河——跟「小何学AI」的"实战日记"逻辑一样，真实经历别人抄不走。

**为什么这个系列很重要**：

你自己身上就有丰富的投资心理学案例。比如：
- 你"不在乎短期涨跌"但每天都看——近视损失厌恶
- 你想"搞清楚为什么涨跌再买股票"——可能是分析瘫痪
- 你买基金金额不大但很在意——心理账户效应

这些都是读者正在经历的事情，共鸣感极强。

**文章结构模板**：

1. **场景**：我今天/这周做了什么投资相关的事
2. **心理独白**：当时我在想什么、感受是什么
3. **觉察**：事后回看，这背后是什么心理偏差在起作用
4. **反思**：我打算怎么调整（或者坦承我调整不了）

**选题来源**：你的日常投资行为——每天看涨跌时的心理波动、决定买入/卖出时的纠结、看到别人赚钱时的心态变化等。

**写作注意事项**：

- 100% 真实，不要美化自己的决策过程
- 承认自己也有心理偏差，不要假装已经克服了
- 数据可以模糊化（不必公开具体金额），但心理过程要详细
- 这个系列跟系列一可以交叉：系列一讲别人的案例，系列四讲自己的案例，同一个心理偏差从两个视角讲

> 🏆 这是你最强的护城河内容。同路人视角+真实心理记录=最高信任度。

## 三、内容策略

### 3.1 启动顺序

1. **首先启动系列一「你为什么亏钱」**——最容易引起共鸣，冷启动效果最好
2. **同步穿插系列四「我的投资心理日记」**——你的真实经历，随时可写
3. **穿插系列二「大师怎么想」**——为伴投埋下预热伏笔
4. **适时加入系列三「投资心理自测」**——作为传播爆点

### 3.2 写作风格要求

- **必须有故事**：每篇都从一个具体场景或案例切入，不要上来就讲理论
- **大白话**：目标读者是投资小白，避免专业术语（或用完立刻解释）
- **有代入感**：让读者在故事里看到自己的影子
- **每篇一个点**：一篇只讲一个心理偏差，讲透，不要贪多
- **篇幅控制**：1500字以内（为未来转视频脚本做准备）

### 3.3 平台选择建议

| 平台 | 适合度 | 说明 |
|------|-------|------|
| 公众号 | ⭐⭐⭐⭐⭐ | 图文长内容首选，沉淀性强，适合系列化输出 |
| 小红书 | ⭐⭐⭐⭐ | 投资理财话题热度高，适合精简版同步发布 |
| 抖音/视频号 | ⭐⭐⭐ | 故事型内容天然适合视频，但制作成本高，后期再启动 |

> 建议初期：公众号为主阵地。如果精力允许，小红书同步发精简版。

### 3.4 现阶段不做的事

- ❌ 短视频制作（先积累图文脚本储备）
- ❌ 硬推伴投产品（先建立内容信任）
- ❌ 追市场热点（坚持系统性输出常青内容）
- ❌ 投资建议或荐股（只讲心理学，不讲操作，规避合规风险）

## 三、灵感捕手（详细设计）

### 3.1 触发方式

你在微信/Discord 上给 bot 发任意消息。

### 3.2 处理流程

1. **接收**：收到文字/链接/截图

### 5.2 投资心理观 初稿 Prompt

```
基于以下素材，为公众号「投资心理观」生成一篇文章初稿。

写作风格要求：
- 用故事讲道理，不要上来就讲理论
- 大白话，避免专业术语（用了要立刻解释）
- 让读者在故事里看到自己的影子
- 一篇只讲一个心理偏差，讲透
- 1500字以内（为未来转视频脚本储备）

文章结构（系列一：你为什么亏钱）：
1. 开头：一个读者能代入的投资场景
2. 故事：一个真实/历史案例
3. 揭示：这是什么心理学原理（大白话解释）
4. 自测：你有没有类似的行为？
5. 一招：一个简单的应对方法

文章结构（系列二：大师怎么想）：
1. 开头：大师的一个经典投资决策场景
2. 故事：他在关键时刻怎么想、怎么做
3. 心理模型：他克服了什么心理偏差
4. 对比：普通散户通常会怎么做
5. 启发：你能学到什么思维方式

文章结构（系列四：我的投资心理日记）：
1. 场景：我今天/这周做了什么投资相关的事
2. 心理独白：当时我在想什么、感受是什么
3. 觉察：事后回看，这背后是什么心理偏差在起作用
4. 反思：我打算怎么调整（或坦承调整不了）
注意：100%真实，不美化决策过程，承认自己也有心理偏差
```
