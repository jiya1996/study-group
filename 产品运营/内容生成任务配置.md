# 自媒体内容生成任务配置

> 每天 19:00 自动执行

---

## 任务描述

为两个公众号账号生成今日内容提案，供用户评审后发布。

---

## 账号一：小何学AI

### 定位
非技术人从"被AI搞懵"到"让AI为我所用"的真实成长记录

### 内容框架（四阶段）

| 阶段 | 内容 |
|------|------|
| 阶段一 | 认知破冰 - 帮读者理解AI核心概念 |
| 阶段二 | 踩坑觉醒 - 真实的AI使用失败和反思 |
| 阶段三 | 方法论建立 - 提示词技巧、AI学习方法 |
| 阶段四 | 实战应用 - 用AI搭产品、做自动化 |

### 内容线比例
- 主线（70%）：AI实战日记
- 副线（30%）：小白方法论

### 选题筛选标准

**必须同时满足**：
1. 非技术读者能理解、会感兴趣
2. 能跟作者"非技术人驾驭AI"定位产生关联
3. 有明确的选题切入角度，不是单纯新闻转述

**优先级排序**：
- 最优：能激发"实战日记"的素材（某个工具更新→可以去试→写体验）
- 次优：能写成"方法论"的素材（新功能/技巧→教读者怎么用）
- 一般：认知破冰类（新概念科普，但要确保非技术人会关心）

## 账号二：投资心理观

### 定位
用故事讲透投资心理学，帮你看见自己投资行为背后的心理陷阱

### 内容系列

| 系列 | 说明 |
|------|------|
| 你为什么亏钱 | 心理陷阱系列（首先启动） |
| 大师怎么想 | 7个投资大师心理模型 |
| 投资心理自测 | 互动系列 |
| 我的投资心理日记 | 个人真实记录 |

### 心理偏差选题库（12+）

损失厌恶、羊群效应、锚定效应、过度自信、沉没成本、确认偏差、禀赋效应、近因偏差、心理账户、处置效应、框架效应、可得性偏差

### 写作风格要求

- 必须有故事：每篇从一个具体场景或案例切入
- 大白话：避免专业术语，用完立刻解释
- 有代入感：让读者在故事里看到自己影子
- 每篇一个点：一篇只讲一个心理偏差，讲透
- 篇幅：1500字以内

### 文章结构模板（系列一：你为什么亏钱）

1. 开头：一个读者能代入的投资场景
2. 故事：一个真实/历史案例
3. 揭示：这是什么心理学原理（大白话）
4. 自测：你有没有类似的行为？
5. 一招：一个简单的应对方法
